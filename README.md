# TP Thailand LINE Recruitment Bot 🇹🇭

A serverless LINE Messaging API bot for TP Thailand recruitment, deployed on Vercel.

## Setup

### Environment Variables (set in Vercel)
- `LINE_CHANNEL_SECRET` — from LINE Developers Console
- `LINE_CHANNEL_ACCESS_TOKEN` — from LINE Developers Console

### Webhook URL
After deploying, set this as your LINE webhook URL:
```
https://<your-vercel-domain>/api/webhook
```

## Features
- Bilingual Thai/English responses
- Menu-driven recruitment flow
- Open positions listing
- Application guidance
- FAQ, office locations, and benefits info
- Follow event greeting
